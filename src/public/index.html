Usage Guide

Yuuka (Discord 秘書 Bot 兼 管理画面) のご利用方法

1. システム概要

『Yuuka』は、Gemini API を活用した Discord 秘書 Bot と Web 管理画面のハイブリッドシステムです。タスク・予定（Googleカレンダー同期）・家計簿などをスマートに管理できます。

2. アカウント登録と初期設定

  • 管理者登録（初期セットアップ）： データベースにユーザーがいない状態で起動すると、初期セットアップ画面が表示されます。Discord ユーザーID、パスワード、および Gemini API キーを入力して管理者を登録します。
  • デフォルトBotの設定： 管理者登録後、Discord Developer Portal で作成した Discord Bot トークンを登録します。
  • 一般ユーザー登録： 管理者が生成した「招待コード」を使用し、新規アカウントを作成します。

3. Discord での操作方法

Bot にメンション、またはDMで話しかけて操作します。

  • タスク管理： 「〇〇をタスクに追加して」「タスク一覧を見せて」「〇〇のタスクを完了して」のように話しかけます。
  • 予定管理： 「明日14時からミーティング」「来週の予定を教えて」でスケジュールが追加され、Google カレンダーに同期されます。
  • 家計簿・支出： 「昼食代800円」「コンビニで日用品を1200円買った」と入力すると、自動で金額やカテゴリを判定して家計簿に記録します。
  • レシート解析： レシート画像を Discord に貼り付けると、Gemini が画像を解析して家計簿へ登録します。

4. Web 管理画面の活用

  • ダッシュボード： 支出の推移グラフやカテゴリ別比率、未完了タスクが一目でわかります。
  • Playbook 管理： AIエージェントに実行させたいブラウザ操作等の自動化手順（Playbook）を記述し、Cron式で定期実行できます。
  • AIレシートスキャナー： 管理画面からもレシート画像をドラッグ＆ドロップして自動家計簿登録が可能です。

Privacy Policy

Yuuka (Discord 秘書 Bot 兼 管理画面) プライバシーポリシー

1. 収集する情報

本システムは、機能提供のために以下の情報を収集・利用します。

  • アカウント情報: Discord ユーザーID、表示名、ログイン用パスワード（ハッシュ化して保存）。
  • AI設定情報: ユーザーが個別に登録した Google Gemini API キー。
  • 外部サービス連携情報: Google Calendar API 連携のための OAuth 認証情報。
  • システム利用データ: タスク、スケジュール、家計簿データ（レシート解析画像を含む）、および自動実行用プレイブック。

2. 情報の利用目的

収集した情報は、以下の目的でのみ使用します。

  • 本サービスにおけるログイン認証およびユーザー識別のため。
  • Gemini API を利用した Discord 秘書 Bot による応答およびレシート画像の解析処理のため。
  • Google カレンダーとのスケジュール同期、および Google Drive へのバックアップ実行のため。
  • 登録されたプレイブックに基づいた自動ブラウジング等の実行のため。

3. データの安全管理

  • Gemini API キー、Discord Token、およびブラウザ自動実行用の外部サービスパスワードなどの機密情報は、暗号化（AES-256-GCM）を施した上でデータベースに保存されます。
  • データの漏洩、紛失、改ざんを防止するため、適切なセキュリティ対策を講じています。

4. 第三者への開示・提供

本システムは、ユーザーの同意がある場合、または法令に基づく場合を除き、収集した個人情報を第三者に提供・開示することはありません。

5. 外部APIの利用

AI機能の処理にあたり、ユーザーが登録したAPIキーを使用して Google Gemini API にリクエストを送信します。送信されたデータの取り扱いは、Googleのプライバシーポリシーに従います。

6. ポリシーの変更

管理者は、本ポリシーを必要に応じて変更することができます。変更後のポリシーは本画面に表示された時点から適用されます。
